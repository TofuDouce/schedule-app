旅手帳工坊

旅遊網站產生器 v2
📦 檔案›
匯入僅接受由「旅手帳模板 v2」或本工坊產出的 .html / .txt 檔。匯入後所有設定會還原到編輯狀態,改完再匯出即可。
✈️ 旅程資料›
這些是網站「第一次開啟」時的預設值;成員在網站上改過之後,以他們改的為準,不會被匯出檔覆蓋。
👥 成員›
點表情符號可換頭像,色塊是成員代表色。
🎨 主題外觀›
📑 自訂分頁›
自訂分頁會出現在網站底部導覽列。每個分頁由「區塊」組成,區塊裡可放文字卡、便箋、收合區塊與功能按鈕。
☁️ 雲端同步(Firebase)›
留空 = 純本機模式。可貼 JSON 或直接從 Firebase 控制台複製的 JS 片段,兩種都認得。
📖 使用說明›
1. 調整左側設定,右側即時預覽。
2. 按「匯出網站」下載 HTML,上傳到 GitHub 即完成部署。
3. 之後要改,把該 HTML 匯入回來繼續編輯。
4. 旅程進行中的記帳、日誌等資料存在各自手機或雲端,匯出檔不會動到它們。
