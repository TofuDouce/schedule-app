資料與部署


    
更新線上網站

    
把目前全部程式碼打包進原始碼，上傳到 GitHub 取代舊的 MyCode.html，線上網站就更新了。（所有程式碼會一起內嵌進檔案）

    

      
      
    


    
備份與搬移（JSON）

    
把全部程式碼存成 JSON 檔保管，或在另一台裝置／瀏覽器還原。還原採合併方式，標題＋內容相同者會自動略過。
